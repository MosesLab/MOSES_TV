MTV = MOSES TV
EGSE = Electronics Ground Support Equipment
MOSES = Multi-Order Solar EUV Spectrograph

MTV_EGSE is an IDL application written by CCK to provide real-time monitoring of
MOSES images during flight.

MTV_BROWSE is for interactively stepping through MOSES data.

MTV is the image display routine. Expects a MOSES raw image file.

MXML is a parser for the XML log file.
